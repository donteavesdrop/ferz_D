# N-Queens Agent System (Stage D)

Распределённое решение задачи о N ферзях с MQTT брокером.

## Быстрый запуск (Windows)

1. Установите [Docker Desktop](https://www.docker.com/products/docker-desktop/).
2. Клонируйте репозиторий.
3. В корне проекта выполните:
docker-compose up -d
Это запустит Mosquitto MQTT брокер на порту 1883.
4. Установите зависимости Python:
pip install -r requirements.txt
5. Запустите приложение:
python main.py
6. В GUI:
- Выберите размер доски и количество ферзей.
- Отметьте галочку «Распределённый (MQTT)».
- Нажмите **«Запустить агентов»** (откроются консольные окна агентов).
- Нажмите **«Найти решения»** — поиск будет выполняться параллельно через MQTT.

## Примечания

- Если брокер недоступен, приложение автоматически перейдёт в локальный режим.
- При закрытии GUI все агенты завершатся.
